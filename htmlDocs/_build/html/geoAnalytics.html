Cosine Similarity = Dot product of df1 and df2 / (Magnitude of df1 * Magnitude of df2)

  • tableName2 – The second data frame containing ‘x’ and ‘y’ columns (point co-ordinates)

geoAnalytics.clustering.DBScan(dataframe, ep, min_sample)[source]
Description:

DBSCAN is a density-based clustering algorithm that groups together data points that are close to each other based on the specified parameters ‘eps’ and ‘min_samples’. The algorithm does not require the number of clusters to be specified in advance and can discover clusters of arbitrary shapes.

Parameters:

  • dataframe – Input data frame containing numerical data to be clustered.

  • (float) (ep) – The maximum distance between two samples for them to be considered as in the same neighborhood.

  • (int) (min_sample) – The number of samples (data points) in a neighborhood for a point to be considered as a core point.

Returns:

Index of the cluster each sample belongs to.

Return type:

ndarray of shape (n_samples,)

geoAnalytics.clustering.affinityPropagation(dataframe, damping_factor, max_iter=300, convergence_iter=15, affinity='euclidean', random_state=None, preference=None)[source]
Description:

Affinity Propagation is a clustering algorithm used to group data points into clusters without requiring the number of clusters as an input. It uses a message-passing approach to determine the exemplars (representative points) for each cluster. The exemplars are data points that best represent the cluster.

Parameters:

  • dataframe – data to be clustered

  • factor (damping) – In the range [0.5, 1.0) -o avoid numerical oscillations when updating values

  • max_iter – maximum number of iterations

  • convergence_iter – default = 15

  • affinity – {‘euclidean’, ‘precomputed’}

  • random_state – int, RandomState instance or None, default=None

  • preference – default = None :points with larger values of preferences are more likely to be chosen as exemplars

Returns:

Index of the cluster each sample belongs to and cluster centers

Return type:

ndarray of shape (n_samples,)
